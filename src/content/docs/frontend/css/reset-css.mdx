## `<img />`要素

[元記事](https://coliss.com/articles/build-websites/operation/css/sensible-defaults-for-img-elements.html)

```css
img {
  max-width: 100%; /* 1 */
  height: auto; /* 1 */
  vertical-align: middle; /* 2 */
  font-style: italic; /* 3 */
  background-repeat: no-repeat; /* 4 */
  background-size: cover; /* 4 */
  shape-margin: 0.75rem; /* 5 */
}
```
