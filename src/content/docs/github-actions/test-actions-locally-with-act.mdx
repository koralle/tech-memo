## なんでactが便利なのか

- ローカルでGitHub Actionsのワークフローをテストできる
- actをMakefileのようなタスクランナーとして扱える

## Install

Mac OSの場合は何も考えずにHomebrewでInstallしちゃおう。

```bash
brew install act
```
