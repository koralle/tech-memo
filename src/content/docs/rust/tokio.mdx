## main関数

tokioに入門した時に最初に書くこれ、

```rust
// main.rs
#[tokio::main]
async fn main() {
    println!("hello");
}
```

こう展開されるらしい

```rust
// main.rs
fn main() {
    let mut rt = tokio::runtime::Runtime::new().unwrap();
    rt.block_on(async {
        println!("hello");
    })
}
```
